HackTheBlob :: Erindale SS

About Us

HackTheBlob is an online hackathon - a sprint-like event in which innovators come up with usable software or hardware with the end goal of creating a functioning product. HackTheBlob aims to use gathered data to improve the lives of students at Erindale Secondary School.
